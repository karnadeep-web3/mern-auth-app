# 🔐 Full Stack Auth App

A full-stack authentication system with JWT-based login.

## 🚀 Features
- User Signup & Login
- Password hashing (bcrypt)
- JWT Authentication
- Protected Routes
- Responsive UI

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB

## 📦 Installation

```bash
npm install
npm start
